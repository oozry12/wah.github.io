但是，请大家注意慎重考虑，原因如下：

1、开发者并非xposed的原创rovo89大神，目前所有的项目都是基于abforce这位开发者

2、xposed框架功能非常强大，是系统层级的修改，代码质量与维护水平影响到整个手机系统的稳定性，可能随时崩溃

3、Android生态圈比较开放自由，鱼龙混杂，在所谓的刷机包内可能嵌入恶意代码，恶意扣费推广之类
